# GoFM Music API

Creating a streaming server in golang.

To run this application, you must have docker installed on your machine
#

## Getting Started

### Launch app

```
make
```

#

## Command useful

### Build project

```
make build
```

### Up project

```
make up
```

### Down container

```
make down
```

### Executing command inside the container

```
make exec
```
